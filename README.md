# Pinboard — To-Do List App

A to-do list styled like a corkboard with pinned index cards — complete with a "DONE" stamp animation and an "unpin" effect when you delete a task.

## Live Demo
[View it here](https://harshvardhan-sachora.github.io/todo-by-harshsachora/todo.html)

## Features
- Add tasks with an optional tag: today, urgent, or someday — each colors the pin differently
- Mark tasks done with a satisfying stamp animation
- Delete tasks with an "unpin" animation
- Filter view: All / Still up / Taken down
- Live counter showing total pinned and completed tasks
- Tasks persist after refresh using localStorage
- Fully responsive card layout

## Built With
- HTML
- CSS
- JavaScript (vanilla — no frameworks or libraries)

## What I Learned
- Managing dynamic UI state (adding, filtering, deleting) in vanilla JavaScript
- Using localStorage to persist data across page refreshes
- Building custom animations (CSS keyframes) for adding/removing items
- Designing a UI with a distinct visual theme instead of a generic template

## Run It Locally
1. Clone this repository
2. Open `todo.html` in any browser — no build steps or dependencies needed
